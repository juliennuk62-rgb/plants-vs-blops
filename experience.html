Politique de confidentialité

Plants vs Blops est un jeu de navigateur gratuit. Dernière mise à jour : 2026.

Données collectées

  • localStorage (stocké sur ton appareil) : ta progression, préférences audio, layout d'éditeur. Pas transmis à un serveur.
  • Firebase Firestore (cloud) : si tu te connectes (Google ou anonyme), ta progression est sauvegardée dans le cloud (cash, plantes, Blops, niveau, stats). Rien d'autre.
  • Firebase Auth : un identifiant unique (UID) est généré. Si connexion Google, ton email est associé à ton compte (visible uniquement par toi).
  • Pas de tracking analytics, pas de cookies, pas de publicité tierce.

Tes droits (RGPD)

  • Accès / export : bouton "Exporter en PDF" dans les paramètres pour voir tes stats.
  • Suppression : bouton "Reset" supprime toutes tes données (localStorage + cloud).
  • Portabilité : te connecter avec Google sur un autre appareil → tu retrouves ta sauvegarde.

Tiers

  • Firebase (Google) : gère l'auth et le stockage cloud. Politique Firebase
  • Discord webhooks (optionnel, si tu actives) : tu configures toi-même l'URL d'envoi. Aucune donnée ne passe par nos serveurs.
  • Assets 3D : licence CC0 (Kenney.nl), réutilisables librement.

Contact

Projet open source, pas de contact commercial. Pour questions / suppression de compte : utiliser le bouton Reset ou supprimer le localStorage du navigateur.
